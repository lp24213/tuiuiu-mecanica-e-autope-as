# Pasta de Imagens

Coloque as seguintes imagens nesta pasta:

## Imagens Necessárias:

1. **logo.png** - Logo da TUIUIÚ AUTO PEÇAS (a imagem que você enviou no chat)
2. **caminhao-oficina.jpg** - Imagem do caminhão Scania vermelho na oficina (que você enviou)
3. **rolamento.jpg** - Imagem do rolamento/bearing (que você enviou)
4. **loja-autopecas.jpg** - Imagem da loja de autopeças (que você enviou)

## Como fazer upload:

1. Salve as imagens que você enviou no chat
2. Renomeie-as conforme os nomes acima
3. Coloque todas nesta pasta `/images/`

O site está configurado para usar essas imagens exatamente como você enviou!
